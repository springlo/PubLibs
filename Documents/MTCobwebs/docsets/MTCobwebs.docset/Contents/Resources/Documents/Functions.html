Deprecations
  • ?=(_:_:)

    If key doesn’t exist, sets its value to expr

    • Deprecation: This will be removed in a future release. Please migrate to static keys and use this gist: https://gist.github.com/radex/68de9340b0da61d43e60

    Note

    This isn’t the same as Defaults.registerDefaults. This method saves the new value to disk, whereas registerDefaults only modifies the defaults in memory.

    Note

    If key already exists, the expression after ?= isn’t evaluated
