Manual scheduling
  • start(runLoop:modes:)

    Schedule this timer on the run loop

    let timer = Timer.new(every: 1.second) {
    print("")
}
//then to start
timer.start()
timer.start(modes: .defaultRunLoopMode, .eventTrackingRunLoopMode)

    By default, the timer is scheduled on the current run loop for the default mode. Specify runLoop or modes to override these defaults.
